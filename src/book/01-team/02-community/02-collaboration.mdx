There are many ways to connect with NUbots, whether you are on the team or not. The [NUbots UoN page](https://www.newcastle.edu.au/research-and-innovation/centre/engineering-built-environment/newcastle-robotics-laboratory/nubots) is a great way to find out information about the team. Other ways to connect with us are the following:

## Repositories

All the NUbots code can be found on [GitHub](https://github.com/NUbots/). To contribute to any NUbots code, you will need to create a GitHub account. A useful GUI for GitHub repositories is [GitKraken](https://www.gitkraken.com/). The main repositories on GitHub for NUbots are the NUbots repository (main codebase) and NUsight2 repository (debugging tool).

## Communication

For communicating with other team members we use [Slack](https://slack.com/intl/en-au/). When you join the team, ask someone to send you an invite to our Slack workspace!
