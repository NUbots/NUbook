This page aims to provide direction and resources on where to go to learn more about various areas of NUbots. There is a mixture of resources available, including external resources and material prepared by NUbots team members.

NUbots workshops may be run live, or you can work through them yourself in your own time.

The majority of resources on this page are free. Some LinkedIn Learning videos are linked here - UoN students can get access to LinkedIn Learning for free.

## General

<summary>NUClear</summary>

The NUbots robot code uses the NUClear framework. Anyone developing robot code needs to learn how to work with NUClear. NUClear was created at NUbots as part of a Software Engineering FYP.

**Reading:**

- [NUbook: NUClear](/system/foundations/nuclear)

**Videos:**

- [YouTube: NUClear Basic Concepts](https://www.youtube.com/watch?v=Bx4yU-eA6bk&t=17m00s)

**Resources:**

## DevTools

<details>
<summary> Web Development </summary>

Web Development is used in the creation of [NUbook](/system/tools/nubook), which uses JavaScript, Gatsby and Tailwind, and [NUsight](/system/tools/nusight), which uses TypeScript, React and MobX. By working on these systems and going through these resources, you will gain valuable real world skills and experience that is often sought out in industry.

**Videos:**

- [egghead.io: MobX](https://egghead.io/courses/manage-complex-state-in-react-apps-with-mobx)

**Tutorials:**

- [The Modern JavaScript Tutorial](https://javascript.info/)
- [Official Node.js Tutorial](https://nodejs.dev/learn)
- [egghead.io: A Beginner's Guide to React](https://egghead.io/courses/the-beginner-s-guide-to-react)
- [Official Webpack Tutorial](https://webpack.js.org/guides/getting-started/)

**Resources:**

- [MDN Web Docs: JavaScript Resources](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- [Official TypeScript Documentation](https://www.typescriptlang.org/docs/handbook/)
- [Official React Documentation](https://reactjs.org/)
- [Official MobX Documentation](https://mobx.js.org/README.html)
- [Official Gatsby Documentation](https://www.gatsbyjs.com/docs/)
- [Official Tailwind Documentation](https://tailwindcss.com/docs)

</details>

<details>
<summary> Documentation </summary>

Since NUbots is a student organisation, high turnover is unavoidable. To help combat the issues this creates, good documentation is key to the success of the team. This includes low level documentation in the form of code comments and high level documentation in the form of NUbook pages.

**Reading:**

- [NUbook: NUbook](/system/tools/nubook)
- [Australian Network on Disability: A Beginner's Guide to Accessible Content](https://www.and.org.au/pages/a-beginners-guide-to-accessible-content.html)

**Tutorials:**

- [Google Technical Writing Course](https://developers.google.com/tech-writing) - free, in-class sessions are open to the public!

</details>
